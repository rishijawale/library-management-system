# 📚 Library Management System

## 🔧 Tech Stack
- Flask (Python)
- MySQL
- HTML, CSS

## 🚀 Features
- User Login
- Book Management
- Issue & Return System
- Overdue Tracking

## ▶️ How to Run
1. Install dependencies:
   pip install flask mysql-connector-python

2. Run:
   python app.py

3. Open:
   http://127.0.0.1:5000
